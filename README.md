# Histopathology Image Analysis

## Introduction
Histopathology is the study of tissue samples to detect diseases such as cancer. This project performs basic image analysis on a small dataset of histopathology images.

## Dataset
- 5–10 histopathology images
- Format: JPG/PNG

## Methodology

### 1. Data Loading
Images were loaded using OpenCV.

### 2. Preprocessing
- Resized to 128x128
- Normalized pixel values

### 3. Visualization
Images displayed using Matplotlib.

### 4. Feature Extraction
- Grayscale conversion
- Histogram analysis
- Edge detection

## Results
- Images successfully processed
- Features extracted from images

## Limitations
- Very small dataset
- No real training performed

## Future Work
- Use larger datasets
- Apply CNN models
